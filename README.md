# 🧙 Mogwarts

Videojuego 2D educativo desarrollado en **Processing (Java Mode)**
sobre el manejo del bullying.

## Controles
| Tecla | Acción |
|-------|--------|
| W/A/S/D | Mover y girar |
| E | Bloquear mensaje |
| Q | Capturar mensaje positivo |
| R | Reportar bullying (necesitas 3 consecutivos) |
| P | Pausar |

## Cómo ejecutar
1. Descarga [Processing](https://processing.org/download)
2. Abre Processing y crea un sketch llamado `game`
3. Crea un tab por cada cada archivo `.pde`, copia y pega el contenido.
4. Presiona ▶

## Tecnologías
- Processing 4 · Java Mode
